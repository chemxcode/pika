## 关于模拟数据正考虑解耦分拆

- home.json 首页用到的 json 文件
- detail.json 详情页用到的 json 文件

# 本地起一台静态服务器
```
npm install -g serve
# -s 参数的意思是将其架设在 Single-Page Application 模式下
# 这个模式会处理即将提到的路由问题
serve -s pika
```
